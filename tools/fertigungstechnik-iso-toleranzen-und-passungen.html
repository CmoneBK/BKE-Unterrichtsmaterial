ISO-Toleranzen und Passungen

Bohrung (Großbuchstaben)



Höchstmaß: GoB = N + ES = mm

Mindestmaß: GuB = N + EI = mm

Toleranz: TB = ES - EI = mm

Fertigungsmaß (Mitte):
MB = (GoB + GuB) / 2 = mm

Welle (Kleinbuchstaben)



Höchstmaß: GoW = N + es = mm

Mindestmaß: GuW = N + ei = mm

Toleranz: TW = es - ei = mm

Fertigungsmaß (Mitte):
MW = (GoW + GuW) / 2 = mm
